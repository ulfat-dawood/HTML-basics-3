Types of HTML Tags

Paired Tags

  1. Have an opening Tag.
  2. Have content / children tags.
  3. Have a closing Tag.
E.g. <p> </p> <ul> </ul>

Unaired Tags

  • Have an opening tag.
  • No content / children tags.
  • No closing tag.
E.g. <br> <hr>

Self-closing Tags

  • Only one tag that closes itself.
  • No content / children tags.
  • Requires attributes.
E.g. <img src="img.png" />     <input type="text" />
